# 🥗 NutriOptima — AI-Powered Nutrition & Diet Management

NutriOptima is a full-stack MERN application that uses AI to provide personalized diet plans, food recognition, calorie tracking, and real-time health monitoring.

---

## ✨ Features

- 🤖 **AI Diet Plans** — Personalized meal plans generated by Groq/Gemini AI
- 📸 **Food Recognition** — Upload a food image to get instant nutrition breakdown
- 📊 **Daily Tracking** — Track calories, macros, and water intake
- 💬 **NutriBot Chatbot** — AI nutrition assistant powered by n8n + Gemini
- 🔔 **Health Alerts** — Real-time alerts based on health conditions and allergies
- 📧 **Email Reminders** — Meal, water, and summary notifications via Gmail
- 🛡️ **Admin Panel** — Manage users, food database, analytics, and moderation

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 18, Vite, Tailwind CSS |
| Backend | Node.js, Express.js |
| Database | MongoDB, Mongoose |
| Primary AI | Google Gemini 2.5 Flash |
| Fallback AI | Groq (Llama 4 Scout) |
| Chatbot | n8n Cloud + Google Gemini |
| Calculations | Python 3 (Mifflin-St Jeor) |
| Email | Nodemailer + Gmail |
| Scheduling | node-cron |

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- MongoDB (local or Atlas)
- Python 3.x
- Git

### 1. Clone the repository
```bash
git clone https://github.com/your-username/nutrioptima.git
cd nutrioptima
```

### 2. Setup environment variables
```bash
cp server/.env.example server/.env
# Edit server/.env and add your API keys
```

### 3. Install dependencies
```bash
# Root dependencies
npm install

# Server dependencies
cd server && npm install

# Client dependencies
cd ../client && npm install
```

### 4. Start the application

**Backend (Terminal 1):**
```bash
cd server
node index.js
```

**Frontend (Terminal 2):**
```bash
cd client
npm run dev
```

Open `http://localhost:3000`
